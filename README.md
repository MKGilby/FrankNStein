Remake of Frank'N'Stein and Frank'N'Stein Rebooted
(from ZX Spectum 48K by Colin Stewart)

Programming language: Pascal (Lazarus 2.2.6 with FPC 3.2.2)
https://www.lazarus-ide.org/
